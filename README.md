# Invoix AI V4
Fuller SaaS MVP: login, Google/Facebook-ready Supabase OAuth, persistent light/dark, business profile, logo/signature upload, editable customers, editable saved invoices, AI/voice invoice generation, premium templates, service reorder, custom columns, QR payment field, invoice-only PDF export, WhatsApp/Gmail/Outlook/email/SMS sharing, reminders, usage limits, Stripe checkout/cancel portal routes.

Env vars: NEXT_PUBLIC_SUPABASE_URL, NEXT_PUBLIC_SUPABASE_ANON_KEY, OPENAI_API_KEY, STRIPE_SECRET_KEY, NEXT_PUBLIC_STRIPE_PRO_MONTHLY_PRICE_ID, NEXT_PUBLIC_SITE_URL.
Run supabase_schema.sql in Supabase.
